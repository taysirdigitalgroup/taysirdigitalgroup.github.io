Guide d'installation (Sources Inconnues)

Nos applications étant distribuées en accès direct sans intermédiaire, votre système Android nécessite d'autoriser l'installation manuelle.

1

Téléchargement

Cliquez sur le bouton de téléchargement pour récupérer le fichier `.apk` sur votre espace de stockage local.

2

Autoriser les Sources Inconnues

Ouvrez le fichier téléchargé. Si une alerte système apparaît, allez dans Paramètres de votre téléphone $\rightarrow$ Sécurité / Applications $\rightarrow$ Cochez ou activez l'option "Autoriser l'installation depuis cette source".

3

Validation & Lancement

Retournez à l'assistant d'installation, cliquez sur Installer, puis ouvrez l'application. Les flux publicitaires se synchroniseront automatiquement.
